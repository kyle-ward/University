# code说明

> image

        存放所有用到的图片

-----

> algorithm.py

        定义了算法类，是本次代码的核心部分

---------------

> main.py：程序的入口

        搭建游戏界面，进行界面内的事件检测

------------------

> chess.py

        定义了博弈树遍历时需要用到的抽象棋盘棋子类，以及一些打印，移动，遍历的方法

-----------------

> constants.py

        存放了所有可能用到的常量，以及评估时需要用到的数据

--------------------

> optimize.py

        定义了一些优化类：

                HistoryTable：历史表类，优化剪枝遍历过程

                Relation：关系类，用于计算棋子的关系价值，优化评估函数

----------------------

> pieces.py

        定义了棋子类，以及将其打印到屏幕上的接口
